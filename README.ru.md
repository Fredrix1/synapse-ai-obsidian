**Synapse AI** превращает твоё хранилище в базу знаний, с которой можно разговаривать. Задавай вопросы, подтягивай заметки в контекст и сохраняй лучшие ответы обратно как постоянные заметки — на модели, которую выбираешь *ты* (Gemini, OpenRouter или любой OpenAI‑совместимый API, включая локальные Ollama / LM Studio), со своим ключом.

> ⚠️ **Бета.** Базовый чат работает и используется ежедневно. Семантический поиск (эмбеддинги), агенты и локальный слой приватности — в планах.

---

## Возможности

- **💬 Чат в боковой панели** — стриминг ответов, остановка на лету, перегенерация, редактирование своих сообщений.
- **🔌 Своя модель** — Gemini, OpenRouter или любой OpenAI‑совместимый эндпоинт (свой `baseURL`, в т.ч. локальные Ollama / LM Studio). Добавляй несколько и переключайся на лету.
- **📎 Контекст из хранилища** — подтягивай заметки в промпт через `[[вики‑ссылки]]`, активную заметку (по переключателю) или автоматический поиск по ключевым словам.
- **💾 Сохранить ответ как заметку** — преврати любой ответ (или выделенный фрагмент) в постоянную заметку со ссылками и frontmatter. Правый клик по сообщению — полное меню.
- **🗂 Автосохранение чата** — диалоги сохраняются как обычные `.md` в твоём хранилище (настраивается, опционально).
- **🌐 Поддержка прокси** — направляй запросы через свой прокси/VPS (для регионов с блокировками).
- **🌍 Локализация** — английский и русский интерфейс.
- **🔒 Контроль на твоей стороне** — заметки и ключ уходят только тому AI‑провайдеру, которого выбрал ты (или через твой прокси). Никакого сервера‑посредника, никакой аналитики.

## С чего начать

**Установка** — скачай `manifest.json`, `main.js`, `styles.css` из последнего [релиза](https://github.com/Fredrix1/synapse-ai-obsidian/releases) в `<хранилище>/.obsidian/plugins/synapse-ai/`, перезапусти Obsidian, включи **Synapse AI** в *Сторонних плагинах*. Или через [BRAT](https://github.com/TfTHacker/obsidian42-brat): *Add beta plugin* → `Fredrix1/synapse-ai-obsidian`.

**Настройка**

1. **Настройки → Synapse AI → API Keys** — добавь ключ (OpenRouter / Gemini).
2. **Models → Add model** — выбери провайдера (или свой OpenAI‑совместимый `baseURL`, например локальный Ollama) и id модели; задай **модель чата по умолчанию**.
3. *(Опционально)* **Proxy URL** для регионов с блокировками; системный промпт, папка чатов, тег по умолчанию, язык.

Открой чат через палитру команд → **«Synapse: Open Chat»**. Напиши вопрос (Enter — отправить), добавь контекст через `[[Название заметки]]`, правый клик по ответу → **Сохранить как заметку**.
